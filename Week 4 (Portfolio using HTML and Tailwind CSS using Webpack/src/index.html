About Me

Get to know me

Hi there! I'm Muhammad Huzaif Amir, a passionate frontend developer and UI designer. I specialize in building clean, responsive websites using HTML, CSS, Tailwind CSS, and tools like Vite and Webpack.

Alongside development, I also design mobile app interfaces in Figma, blending usability with modern UI trends. From web layouts to mobile screens, I enjoy turning ideas into visuals that feel intuitive and engaging.

I've recreated Shopify themes like Craft, Dawn, and Tinker to sharpen my skills in layout structuring, responsiveness, and component-based design.

My Services

I replicate premium themes such as Shopify’s Craft, Dawn, and Tinker into clean, maintainable HTML/CSS or Tailwind code — perfect for design-to-code handoffs, portfolio pieces, or frontend learning.
